# 发布说明

## 1.1.0 - 2026-06-30

### 本次实现

- 将单一云端配置升级为可创建、切换、删除的多 `Backup Profile` 管理模型。
- 重做应用内云端配置页，支持直接保存、切换、粘贴和删除 S3 / OSS 配置。
- 优化同步状态提示，播放页和配置页都会显示可关闭的悬浮通知，不再推动界面布局。
- 新增云端歌曲按需下载进度与失败状态反馈，避免重复触发下载。
- 远端歌曲索引改为按 Profile 隔离，备份和同步结果会写回当前配置的状态。
- 下线 `.env` 与 secrets plugin 配置流程，S3 / OSS 信息改为完全通过应用内设置维护。

## 1.0.1 - 2026-06-29

### 本次实现

- 新增 S3 兼容对象存储的云端歌单同步与备份能力。
- 完成七牛云 Kodo 的 S3 兼容配置与接入验证。
- 支持从对象存储同步云端歌曲清单。
- 支持将云端歌曲下载到本地，并进行校验。
- 支持应用重新打开后恢复上一次播放的歌曲、播放进度和循环模式。
- 支持锁屏界面和通知栏中的媒体播放控制。
- 支持显示已下载本地音频文件的内嵌封面。
